== Description ==

It is a simple Hotspot addon for  addon for elementor.After installing this plugin you will get a HOt spot widget in the elementor widget area.

[youtube https://www.youtube.com/watch?v=gNYxnTMu_EI]


== Features ==
* Customizable hotspot 
* Add map locator spot
* Fully responsive 
* Possible to add own colors,fonts and styles
